# Art Assets & AI Generation Prompts

This spec contains the exact prompts you will use in Midjourney/DALL-E to generate assets without backgrounds to composite our scene.

## 1. Prompt Consistency Anchor
The foundational style for all images generated for this game MUST use this incredibly specific base description to guarantee absolute uniformity across all outputs:
> `aesthetic of a stylized premium 3D mobile simulation game, smooth clay-like digital textures, matte finish, soft global illumination lighting, crisp geometric architecture, flat blocks of pastel and vibrant accent colors, minimal surface details, clean untextured surfaces. Avoid specific university logos, avoid specific subject text, use abstract generic details for posters and chalkboards.`

## 2. The 2x2 Consistency Grids (Avatar Creator)
We need all hairstyles and outfits generated within the *same* image to ensure the AI uses the exact same art style/lighting constraint. It must generate with a completely transparent background so no editing is required. **Crucially: All four quadrants must be perfectly centered at the exact same scale, with the subject's waist anchored squarely on the bottom edge of their grid box.**
